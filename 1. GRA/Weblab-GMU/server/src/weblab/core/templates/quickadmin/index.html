Quickadmin

The quickadmin is a tool for being able to easily track the usage of WebLab without dealing with authentication.

Provide a quickadmin_token configuration variable with a secret, and you will only be able to see these pages by providing ?token=secret

This way, you can bookmark this page in your mobile phone, screens and so on to know what's going on. Of course, you will never be able to perform admin tasks such as adding or removing users or so with this token. It's only intended for visualization tasks
